The textbook covers ten chapters:

1. **The Writing System** — 18 base letters, 19 sound symbols, the chunk system, substitution rules
2. **Sounds & Phonology** — the floating sound modifier system
3. **Punctuation** — enclosing marks, pairs, and what doesn't exist in Cythora
4. **Core Vocabulary** — the 14 clan greetings, pronouns, conjunctions, prepositions, colors
5. **Grammar Fundamentals** — SVO structure, negation, questions, imperatives, relative clauses

8. **Passive Voice** — all passive forms across every tense and modal
9. **Special Constructions** — keep + verb-ing, used to, just + verb, do/did emphasis
10. **The Fourteen Clans** — emblems, Cythora script names, and slogans for all 14 clans

## The Writing System — Quick Reference

Cythora has **18 base letters** and **19 sound symbols**. Each written unit (a *chunk*) is a spatial cluster — one central base letter with up to three sound modifier glyphs orbiting around it.

**Chunk priority:** 4-char → 3-char → 2-char  
**Letter substitutions:** C→S · Y→I · Z→J · U→O · W→V · X/Q→K  
**H-rule:** H has no base letter — it exists only as a sound symbol and can never begin a chunk.

## Grammar — Quick Reference

| Feature | Rule |
|---------|------|
| Word order | SVO |
| Verb infinitive | ends in `-dra` (e.g. *klodra* = to walk) |
| Negation | `khed` after object, or after helping clause |
| Articles | none — Cythora has no a / an / the |
| Questions | same word order, interrogative word at front, enclosed in `? … ?` |
| Requests | `osorashi` at end of sentence (replaces "please") |
